# CHECKLIST.md

## 1. 每次改代码前检查

- [ ] 明确本次只做一个阶段目标
- [ ] 确认不会破坏现有功能
- [ ] 确认不提交真实密钥
- [ ] 确认不提交 node_modules / venv / .env / 数据库文件
- [ ] 确认是否涉及数据库结构变化
- [ ] 如果涉及数据库变化，说明是否需要删除 `backend/data/app.db`

## 2. 前端检查

- [ ] `npm run build` 通过
- [ ] 路由能正常访问
- [ ] 菜单能正常跳转
- [ ] 页面 loading 状态正常
- [ ] 页面 empty 状态正常
- [ ] 接口错误有 message 提示
- [ ] 表格字段显示正常
- [ ] 表单校验正常
- [ ] 操作按钮有反馈
- [ ] 页面文案统一为 AI短剧 / 短剧项目

## 3. 后端检查

- [ ] `python -m compileall app` 通过
- [ ] `uvicorn app.main:app --reload --port 8000` 能启动
- [ ] `GET /api/health` 正常
- [ ] 新增接口能在 `/docs` 中看到
- [ ] 查询接口支持分页或基础筛选
- [ ] 详情不存在时返回 404
- [ ] 修改接口能更新 updated_at
- [ ] 删除优先软删除
- [ ] mock fallback 正常
- [ ] 不影响已有接口

## 4. AI 功能检查

- [ ] `AI_PROVIDER=mock` 可运行
- [ ] `AI_PROVIDER=deepseek` 时才调用真实模型
- [ ] AI 失败时有 fallback
- [ ] AI 输出结构能被前端稳定渲染
- [ ] 分镜结果包含人物一致性提示词
- [ ] 广告素材结果包含 hook / CTA / A/B 测试建议
- [ ] 本地化结果区分中文审核稿和目标语言稿
