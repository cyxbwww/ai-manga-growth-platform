# AI漫剧出海增长平台 Demo 讲解稿

## 双语输出设计

这次优化解决的是“目标市场选择日本、目标语言选择日文后，AI 结果直接变成日文，团队内部不方便审核”的问题。

核心设计是：接口原字段继续保留中文，同时新增 `bilingual` 字段保存中文版本和目标语言版本。

- 中文：用于内部审核、团队协作、面试演示和内容策略复盘。
- 目标语言：用于海外投放、本地化生产、广告文案测试和提示词复制。
- 前端：在内容策划、剧本打磨、分镜制作、广告素材模块提供“中文 / 目标语言”切换。
- 兼容旧数据：历史记录没有 `bilingual` 时，继续按原字段展示，不影响旧数据查看。
- AI 兜底：DeepSeek 正常时返回双语 JSON；未配置、失败或 JSON 解析失败时，mock fallback 也返回双语结构。

演示顺序建议：

1. 打开内容策划，选择“日本 + 日文”，点击生成，先展示中文审核版，再切换到日文/目标语言版。
2. 打开剧本打磨，说明 AI 不是简单生成，而是优化节奏、冲突、反转和海外表达。
3. 打开分镜制作，切换语言后复制 visualPrompt 或 motionPrompt，强调提示词可直接进入生产链路。
4. 打开广告素材，切换语言后复制当前语言广告文案，说明同一素材支持内部审核和海外投放双场景。
5. 打开本地化模块，展示中文原文、直译、本地化改写，强调“不是翻译，而是海外表达适配”。
